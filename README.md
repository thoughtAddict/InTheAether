
InTheAether
===

Online super chat and collaboration application.
 
This project will utilize...

    Node JS
    
    Express JS
    
    Now JS

    MongoDB
    
    Mongoose ORM

    jQuery
        front-end 

...with the end-purpose to provide a place for "general banter" for friends across the globe.
