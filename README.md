
InTheAether
===

Online super chat and collaboration application.  (---*** Heavily under development *** ---)
 
This project will utilize...

        Node JS
    
        Express JS
    
        Now JS

        MongoDB
    
        Mongoose ORM

        jQuery

        Cloud9 IDE

...with the end-purpose to provide a place for "general banter" for friends across the globe.
